0x00 引言

各种选择器可以组合使用。

0x01 基本概念

行内元素 ，块级元素<div> </div>。

CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明：

selector {declaration1; declaration2; ... declarationN }

如果值为若干单词，则要给值加引号：

p {font-family: "sans serif";}

层叠次序：

浏览器缺省设置
外部样式表
内部样式表（位于 <head> 标签内部）
内联样式（在 HTML 元素内部）

选择器的分组：

h1,h2,h3,h4,h5,h6 {
  color: green;
  }

继承：

body  {
 font-family: Verdana, sans-serif;
 }

整个网页的字体就是 Verdana

0x02 选择器

id 选择器和类选择器都能用作派生选择器

派生选择器（元素选择器，类型选择器）

li strong {
    font-style: italic;
    font-weight: normal;
  }

id 选择器

#red {color:red;}
#green {color:green;}

类选择器

.center {text-align: center}

属性选择器

[title]
{
color:red;
}

属性和值选择器

[title=W3School]
{
border:5px solid blue;
}
选择器 描述
[attribute] 用于选取带有指定属性的元素。
[attribute=value] 用于选取带有指定属性和值的元素。
[attribute~=value] 用于选取属性值中包含指定词汇的元素。
[attribute =value] 用于选取带有以指定值开头的属性值的元素，该值必须是整个单词。
[attribute^=value] 匹配属性值以指定值开头的每个元素。
[attribute$=value] 匹配属性值以指定值结尾的每个元素。
[attribute*=value] 匹配属性值中包含指定值的每个元素。

通配符选择器

* {color:red;}

后代选择器

h1 em {color:red;}

上面这个规则会把作为 h1 元素后代的 em 元素的文本变为 红色。

0x03 class与id

class 选择器用来公用，id 选择器用于操作。

0x04 html引用css方法

1、直接在div中使用css样式制作div+css网页
2、html中使用style自带式

<style type="text/css"> 
<!-- 
.ceshi {font-size:14px; color:#FF0000;}/*这里是设置CSS的样式内容*/ 
--> 
</style>

3、使用@import引用外部CSS文件

<style type="text/css"> 
<!-- 
@import url(wcss.css);/*这里是通过@import引用CSS的样式内容*/ 
--> 
</style>

4、使用link引用外部CSS文件（推荐此方法）

<link rel="stylesheet" href="wcss.css" type="text/css" />

0x05 examples

#E F{color:red;}          选择 E 元素后所有 F 后代元素           
#E > F{color:red;}       E > F 表示选择E元素的所有子F元素，只选择一代
#E + F{color:inherit;} 相邻兄弟选择器可以选择紧接在另一元素后的元素，而且他们具有一个相同的父元素
#E F:nth-child(2){color:red}

E+F表示HTML中紧随E的F元素。

nth-child是个伪类的用法，如F:nth-child(2)就表示在F的父元素中选择位居第二位的F。
